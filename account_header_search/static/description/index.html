This add-on is a funtional plug-in that creates a fixed filter bar in the accounting module for list views. It allows users to customize their search with a single click instead of navigating to the conventional search bar. Additionally, users can select specific time periods according to their needs, and the records will update automatically.

Description

To simplify the record selection process, an in-page filter is added to the top of the list view. During regular operations, this filter can remain unselected to show all records. However, when activated, it automatically filters records based on a specified date range. Users can install this module independently without needing additional libraries, as it uses JavaScript-based selectors and native fields for the searches. If no records are found, the filter automatically clears the list view.
