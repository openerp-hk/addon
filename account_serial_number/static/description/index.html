Row number control in account

This add-on is a functional plug-in that allows users to control the visibility of row numbers in the account module. Additionally, in edit mode, the row numbers follow the drag-and-drop order without losing any information.

Description

To give users the flexibility to add row numbers to views by toggling the control button. Controlling row number visibility is just a click away. This module has been tested with Odoo version 17 and is expected to be compatible with versions down to Odoo v15.
